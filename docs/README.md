# **Проект Банковское приложение.**
## Описание данное приложение создано с целью обучения на данный момент. 
#### ***Примечание это пример подобных приложений созданный на основе учебного материала для обучения.***

## Нововведение и функции:
###  Нововведение для пользователей
1. Была введена возможность ввести номер карты или номер счёта.
2. Была введена возможность ввести дату.
###  Нововведение технические не предназначенные для глаз пользователя 
1. Была введена функция маска для номера карты.
2. Была введена функция маска для номера счёта.
3. Была введена функция общего приёма для номера карты номера счёта от пользователя.
4. Была введена функция приёма даты от пользователя.
5. Была введена функция фильтрация список данных по ключу 'state'.
6. Была введена функция сортировки список данных по дате.
7. Была введена функция генератор номеров банковских карт.
8. Была введена функция декоратора логинов.
9. Была введена функция читает JSON-файл.
10. Была введена функция конвертирует сумму транзакции в рубли.
11. Была введена функция дату в формате ГГГГ-ММ-ДД даты и времени.
12. Была введена функция конвертированную сумму через внешний API.
####  Дальнейшие функции будут описаны в последующих обновлениях.

Для генераторов пришлось использовать встроенную библиотеку typing для корректной работы были взяты некоторые функции Any, Dict, Generator, Iterator, List
Для декоратора пришлось использовать встроенную библиотеку typing для корректной работы были взяты некоторые функции Any, Callable, Dict, Optional, Tuple
Из встроенной библиотеки functools функции wraps для метаданных. 
Для тестирование кода декоратора я использовал встроенную системную библиотеку os, библиотеку typing для корректной работы были взяты некоторые функции Any, Generator    

## Инструкция по настройке линтеров для данного проекта

Создайте файл с названием ".flake8" и в нём введите следующие
```
[flake8]
max-line-length = 119
ignore = E402, E251, E203
exclude = .git, .venv, __pycache__, venv

```
Вводите в файл pyproject.toml, если там не прописано, то пропишите следующие.
```
[tool.black]
# Максимальная длина строки
line-length = 119
# Файлы, которые не нужно форматировать
exclude = '''
(
  /(
      \.eggs         # Исключить несколько общих каталогов
    | \.git          # в корне проекта
    | \.hg
    | \.mypy_cache
    | \.tox
    | \.venv
    | dist
  )/
  | foo.py           # Также отдельно исключить файл с именем foo.py
                     # в корне проекта
)
'''


[tool.isort]
# максимальная длина строки
line_length = 119


[tool.mypy]
disallow_untyped_defs = true
warn_return_any = true
ignore_missing_imports = true
exclude = "venv"

```
И все зависимости что есть на данный момент будут прописаны.
