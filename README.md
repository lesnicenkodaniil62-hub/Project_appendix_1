# Проект Банковское приложение

## Описание:

Проект Банковское приложение - это веб-приложение на Python для управления задачами и проектами.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/lesnicenkodaniil62-hub/Project_appendix_1
```
2. Установите зависимости:
```
poetry init
```
## Использование:

1. Откройте приложение в вашем веб-браузере.
2. Введите номер счёта и дату.
3. Дальнейшие функции будут доступны с последующими обновлениями.
